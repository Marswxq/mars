# HTML XML 转义

| 显示  | 说明     | 实体名称       | 十进制编号     |

| ` ` | 半角空格   | `&ensp;`   | `&#8194;` |
| ` ` | 全角空格   | `&emsp;`   | `&#8195;` |
| ` ` | 不断行的空格 | `&nbsp;`   | `&#160;`  |
| `<` | 小于     | `&lt;`     | `&#60;`   |
| `>` | 大于     | `&gt;`     | `&#62;`   |
| `&` | &符号    | `&amp;`    | `&#38;`   |
| `"` | 双引号    | `&quot;`   | `&#34;`   |
| `©` | 版权     | `&copy;`   | `&#169;`  |
| `®` | 已注册商标  | `&reg;`    | `&#174;`  |
| `™` | 商标（美国） | `&trade;`  | `&#8482;` |
| `×` | 乘号     | `&times;`  | `&#215;`  |
| `÷` | 除号     | `&divide;` | `&#247;`  |
