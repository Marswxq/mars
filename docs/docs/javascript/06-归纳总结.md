# 归纳总结

## span内容溢出显示省略号，鼠标悬停显示完整内容

html:

```html
<span class="ellipsis" :title="data.span">{{data.span}}</span>
```

css:

```html
.ellipsis{
    white-space:nowrap;
    text-overflow:ellipsis;
    overflow:hidden;
    width: 20em; /* 20个字 */
    display:inline-block;
}
```

** 注意: ** 如`<span>`标签样式不生效，使用`<div class="ellipsis">`包裹并传递样式。
